**–°–æ—Ä—Ç–∏—Ä—É–µ–º –ø—Ä–µ–¥–º–µ—Ç—ã –ø–æ —É–¥–µ–ª—å–Ω–æ–π —Å—Ç–æ–∏–º–æ—Å—Ç–∏ (—Ü–µ–Ω–∞/–≤–µ—Å) –≤ —É–±—ã–≤–∞—é—â–µ–º –ø–æ—Ä—è–¥–∫–µ. –ë–µ—Ä–µ–º –ø–æ–ª–Ω–æ—Å—Ç—å—é –ø—Ä–µ–¥–º–µ—Ç—ã —Å –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–π —É–¥–µ–ª—å–Ω–æ–π —Å—Ç–æ–∏–º–æ—Å—Ç—å—é, –ø–æ—Å–ª–µ–¥–Ω–∏–π –ø—Ä–µ–¥–º–µ—Ç –±–µ—Ä–µ–º —á–∞—Å—Ç–∏—á–Ω–æ.**

#### –ü–æ—á–µ–º—É —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç?

**–õ–µ–º–º–∞ –æ –∂–∞–¥–Ω–æ–º –≤—ã–±–æ—Ä–µ:**
–ü—É—Å—Ç—å –ø—Ä–µ–¥–º–µ—Ç —Å –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–π —É–¥–µ–ª—å–Ω–æ–π —Å—Ç–æ–∏–º–æ—Å—Ç—å—é ‚Äî `i`. –¢–æ–≥–¥–∞ –æ–ø—Ç–∏–º–∞–ª—å–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ —Å–æ–¥–µ—Ä–∂–∏—Ç –ø—Ä–µ–¥–º–µ—Ç `i` (—Ö–æ—Ç—è –±—ã —á–∞—Å—Ç–∏—á–Ω–æ).

**–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:**
1. –ï—Å–ª–∏ –ø—Ä–µ–¥–º–µ—Ç–∞ `i` –Ω–µ—Ç –≤ –æ–ø—Ç–∏–º–∞–ª—å–Ω–æ–º —Ä–µ—à–µ–Ω–∏–∏ `O`, –≤–æ–∑—å–º–µ–º –¥—Ä—É–≥–æ–π –ø—Ä–µ–¥–º–µ—Ç `j` –∏–∑ `O`

**–ó–∞–º–µ—á–∞–Ω–∏–µ:** –≠—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç **—Ç–æ–ª—å–∫–æ –¥–ª—è –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–π –≤–µ—Ä—Å–∏–∏**! –î–ª—è –¥–∏—Å–∫—Ä–µ—Ç–Ω–æ–π –≤–µ—Ä—Å–∏–∏ (0-1) –∂–∞–¥–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –º–æ–∂–µ—Ç –±—ã—Ç—å –¥–∞–ª–µ–∫–æ –Ω–µ–æ–ø—Ç–∏–º–∞–ª–µ–Ω (—Å–º. –Ω–∏–∂–µ).

### 4. –ê–ª–≥–æ—Ä–∏—Ç–º –•–∞—Ñ—Ñ–º–∞–Ω–∞ (Huffman Coding)

#### –ó–∞–¥–∞—á–∞
–î–∞–Ω —Ç–µ–∫—Å—Ç —Å —á–∞—Å—Ç–æ—Ç–∞–º–∏ —Å–∏–º–≤–æ–ª–æ–≤. –ü–æ—Å—Ç—Ä–æ–∏—Ç—å –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–π –ø—Ä–µ—Ñ–∏–∫—Å–Ω—ã–π –∫–æ–¥ (–Ω–µ—Ç –∫–æ–¥–∞ –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –ø—Ä–µ—Ñ–∏–∫—Å–æ–º –¥—Ä—É–≥–æ–≥–æ –∫–æ–¥–∞) –¥–ª—è –º–∏–Ω–∏–º–∏–∑–∞—Ü–∏–∏ —Å—Ä–µ–¥–Ω–µ–π –¥–ª–∏–Ω—ã –∫–æ–¥–æ–≤–æ–≥–æ —Å–ª–æ–≤–∞.

#### –ñ–∞–¥–Ω–∞—è —Å—Ç—Ä–∞—Ç–µ–≥–∏—è
**–ù–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ –æ–±—ä–µ–¥–∏–Ω—è–µ–º –¥–≤–∞ —É–∑–ª–∞ —Å –Ω–∞–∏–º–µ–Ω—å—à–∏–º–∏ —á–∞—Å—Ç–æ—Ç–∞–º–∏. –ü–æ–≤—Ç–æ—Ä—è–µ–º, –ø–æ–∫–∞ –Ω–µ –æ—Å—Ç–∞–Ω–µ—Ç—Å—è –æ–¥–∏–Ω —É–∑–µ–ª (–∫–æ—Ä–µ–Ω—å –¥–µ—Ä–µ–≤–∞).**

#### –ü–æ—á–µ–º—É —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç?

**–õ–µ–º–º–∞ –æ –∂–∞–¥–Ω–æ–º –≤—ã–±–æ—Ä–µ:**
–ü—É—Å—Ç—å `a, b` ‚Äî –¥–≤–∞ —Å–∏–º–≤–æ–ª–∞ —Å –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–º–∏ —á–∞—Å—Ç–æ—Ç–∞–º–∏. –¢–æ–≥–¥–∞ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–π –∫–æ–¥, –≤ –∫–æ—Ç–æ—Ä–æ–º `a` –∏ `b` –∏–º–µ—é—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—É—é –≥–ª—É–±–∏–Ω—É –∏ —è–≤–ª—è—é—Ç—Å—è —Å–∏–±–ª–∏–Ω–≥–∞–º–∏ –≤ –¥–µ—Ä–µ–≤–µ –∫–æ–¥–æ–≤.

**–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:**
1. –í –æ–ø—Ç–∏–º–∞–ª—å–Ω–æ–º –¥–µ—Ä–µ–≤–µ `a` –∏ `b` –Ω–∞ –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–π –≥–ª—É–±–∏–Ω–µ (–∏–Ω–∞—á–µ –º–æ–∂–Ω–æ —Å–æ–∫—Ä–∞—Ç–∏—Ç—å –∫–æ–¥—ã)
2. –ï—Å–ª–∏ `a` –∏ `b` –Ω–µ —Å–∏–±–ª–∏–Ω–≥–∏ —Å –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–º–∏ —á–∞—Å—Ç–æ—Ç–∞–º–∏, –º–æ–∂–µ–º –∏—Ö –ø–µ—Ä–µ—Å—Ç–∞–≤–∏—Ç—å
3. –û–±—â–µ–µ –∏–∑–º–µ–Ω–µ–Ω–∏–µ –¥–ª–∏–Ω—ã: `(len_a + 1) + (len_b + 1) - len_a - len_b = 2`, –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ —á–∞—Å—Ç–æ—Ç–∞–º
4. –¢–∞–∫ –∫–∞–∫ `a, b` –∏–º–µ—é—Ç –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–µ —á–∞—Å—Ç–æ—Ç—ã, —ç—Ç–∞ –∑–∞–º–µ–Ω–∞ –º–∏–Ω–∏–º–∏–∑–∏—Ä—É–µ—Ç –ø—Ä–∏—Ä–æ—Å—Ç

**–û–ø—Ç–∏–º–∞–ª—å–Ω–∞—è –ø–æ–¥—Å—Ç—Ä—É–∫—Ç—É—Ä–∞:**
–ü–æ—Å–ª–µ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è `a` –∏ `b` –ø–æ–ª—É—á–∞–µ–º –Ω–æ–≤–æ–µ –¥–µ—Ä–µ–≤–æ, –∏ –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–π –∫–æ–¥ –¥–ª—è –æ—Å—Ç–∞–≤—à–∏—Ö—Å—è —Å–∏–º–≤–æ–ª–æ–≤ + –æ–±—ä–µ–¥–∏–Ω–µ–Ω–Ω–æ–≥–æ —É–∑–ª–∞ —Å—Ç—Ä–æ–∏—Ç—Å—è —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ.

#### –í—Ä–µ–º–µ–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å
- **O(n log n)** - –≥–¥–µ n ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤
- **O(m)** - –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ —Ç–µ–∫—Å—Ç–∞, –≥–¥–µ m ‚Äî –¥–ª–∏–Ω–∞ —Ç–µ–∫—Å—Ç–∞
- **O(m)** - –¥–µ–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ

#### –†–µ–∞–ª—å–Ω—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ç–µ—Å—Ç–∞
```

–î–µ—Ä–µ–≤–æ –•–∞—Ñ—Ñ–º–∞–Ω–∞ (–≤–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è):
```
